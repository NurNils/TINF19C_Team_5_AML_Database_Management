Help and info page

Come here if you have any problems according the propper usage of the AML NoSQL-Database


The AML-NoSQL-Database-Management is used to upload AML files onto a NoSQL-Database to provide a unified and central solution to manage AML files.
AML is a file format developed as an exchange format with an open standard.
From the user's point of view, the GUI should have the following functionalities:
  1. Upload AML files from the user's computer.
  2. Download files to the computer that have been uploaded.
  3. Edit saved files with a dialog window and upload the changed version.
  4. Delete saved files from a list of files.
  5. Search for existing AML files with the file name.

Main page

Let's move to the most important part of the website: the main site.



Here you can find the heart of our software: the AML NoSQL Database. Starting on the upper right side you can find two buttons:



Both are used to get add new AML files into the AML NoSQL-Database.

Uploading a file

The left button which sais is used to upload .aml files from your computer.

If you click this button you should see a little popup which sais 'Upload file. Here you can upload your AML file.'.



Under this texts on the left you can see a little button with 'Choose File'. Right next to the button on the right side is an info text to show your upload status. By default it sais 'No file chosen'. In the bottom right corner you will see two buttons. The left one sais 'Cancel' and can be used to abort the upload process. The right one which is greyed and blocked by default sais 'Upload'. To make use of the upload button, you have to press the 'Choose File' button first. Pressing it will open your local file system. You can now search your local file system for files you want to upload. Just select them and press confirm. How exactly your file system as well as the confirmation process looks depoends on your used Operation-System. After you have selected a file you will see that your selected file is shown at the spot, where 'No file chosen' stood before.



Use this to make sure you have chosen the right file. If you are certain that you selected everything correctly, you can finally press the 'Uplaod' button. You will be redirected to the main page and the selected file will be uploaded. Make sure you see a little altert box at the bottom of the site which sais 'File was uploaded succesfully'.



Note that it is not possible to upload files which have the same name like an already existing one.


Creating a file

Alternatively you can also choose to make use of the button.

As the name sais, it is used to create .aml files directly in the AML-NoSQL Database. Clicking it will also show you an alert box which looks very similar to the one where you can upload.



Like in the upload process there are 'Cancel' and 'Upload' buttons with the same functionallities as in the upload process. This time you have an input line which sais 'Name'. Use this to declare a name but keep in mind to choose a unique name because otherwise you will get an error. As you can see on the right side of the inputline, you dont have to to write .aml as file type. The software does this automatically for you. The big Textfield down under is used to write code for your .aml file. It even supports syntax highlighting. When you are done writing you can confirm by clicking the 'Upload' button.



You will be redirected to the main page and your file will appear at the table. Again make sure that you see the little alert box that tells you that the creation was succesfull.




Finding a file

When you have uploaded several files to the database, finding specific items can get difficult. To help you finding the files you are searching for, AML-NoSQL provides a searchbar on the left top of the Database to specify filterwords.



You can type all kinds of keywords for example numbers to search for the date, ID or size or type characters to search for the name of the file. A combination of numbers and characters is possible as well.



Therefore you will be able to filter out files with attributes you don't want.


Using the Table

The table is the heart of the main page. All uploaded .aml files will appear here.




Sorting files

In order to find them easier you can sort the table. Therefore you shall use the sort bar.



To do this, just press one of the categories at the table header. You can filter ascending or descending by either 'ID', which is an identifier given by the software to identify uploaded files, Name, which is the name you have chosen for your file, 'Size', which is the size of your created file and 'Date', which is the date of creation (not upload!) of your file. To switch between ascending and descending just click the attribute of choise once again.




By clicking it a third time you can stop sorting.


Edit, delete or delete a file

The item on the right side of the table header sais 'Actions'.



Here you can delete, download or edit files. To edit a file you must click the symbol.
You will then be redirected to a alertbox which looks like the 'Create file' one. You can now change the name and edit the code of the file. Of course you can 'Cancel' the operation if you dont want to do any changes. If you are done editing just press 'Upload'.

To download files you can use this button.
The download will start automatically.

The last option is to delete files. To delete a file press the button.
To make sure you do not delete a file on accident, you will get a second instance confirmation before the file is deleted.



You will have to press 'OK' to confirm the deletion process.


Paging

At the bottom right corner you can see the paging operations.



For reasons of user experience and performance only 10 items are shown per page.



Anyways you can set this number up tp 200. Just click on the number right next to 'Items per page' ans select your number of choice.



On the right side of this selection you can see the amount of items which are shown per page. Last but not least in the right corner you can see two arrows.



You can use them to navigate between multiple table pages so you dont have to show too many items per page.

Hilfe- und Infoseite

Kommen Sie hierher, wenn Sie Probleme mit der korrekten Nutzung der AML NoSQL-Datenbank haben


Das AML-NoSQL-Datenbank-Management wird verwendet, um AML-Dateien in eine NoSQL-Datenbank zu laden, um eine einheitliche und zentrale Lösung zur Verwaltung von AML-Dateien bereitzustellen.
AML ist ein als Austauschformat entwickeltes Dateiformat mit einem offenen Standard.
Aus Sicht des Benutzers sollte die GUI folgende Funktionalitäten haben:
  1. Hochladen von AML-Dateien vom Computer des Benutzers.
  2. Herunterladen von Dateien auf den Computer, die hochgeladen wurden.
  3. Gespeicherte Dateien mit einem Dialogfenster bearbeiten und die geänderte Version hochladen.
  4. Gespeicherte Dateien aus einer Liste von Dateien löschen.
  5. Mit dem Dateinamen nach vorhandenen AML-Dateien suchen.

Hauptseite

Kommen wir nun zum wichtigsten Teil der Website: der Hauptseite.



Hier finden Sie das Herzstück unserer Software: die AML NoSQL-Datenbank. Beginnend auf der oberen rechten Seite finden Sie zwei Buttons:



Beide werden verwendet, um neue AML-Dateien in die AML NoSQL-Datenbank einzufügen.

Hochladen einer Datei

Die linke Schaltfläche, die heißt, dient zum Hochladen von .aml-Dateien vom eigenen Rechner.

Wenn Sie auf diese Schaltfläche klicken, sollte ein kleines Popup-Fenster erscheinen, in dem es heißt: "Datei hochladen". Hier können Sie Ihre AML-Datei hochladen.'.



Unter diesen Texten sehen Sie auf der linken Seite eine kleine Schaltfläche mit 'Datei wählen'. Rechts neben der Schaltfläche befindet sich ein Infotext, der den Upload-Status anzeigt. Standardmäßig steht dort "Keine Datei ausgewählt". In der unteren rechten Ecke sehen Sie zwei Schaltflächen. Die linke heißt "Abbrechen" und kann verwendet werden, um den Upload-Vorgang abzubrechen. Die rechte Schaltfläche die standardmäßig ausgegraut und blockiert ist, heißt "Hochladen". Um die Schaltfläche "Hochladen" zu verwenden, müssen Sie müssen Sie zuerst die Schaltfläche "Datei auswählen" drücken. Wenn Sie darauf drücken, wird Ihr lokales Dateisystem geöffnet. Sie können Sie können nun Ihr lokales Dateisystem nach Dateien durchsuchen, die Sie hochladen möchten. Wählen Sie diese einfach aus und drücken Sie bestätigen. Wie genau Ihr Dateisystem sowie der Bestätigungsvorgang aussieht, hängt von dem von Ihnen verwendeten Operationssystem ab. Nachdem Sie eine Datei ausgewählt haben, sehen Sie, dass Ihre ausgewählte Datei an der Stelle angezeigt wird, an der vorher 'Keine Datei ausgewählt' vorher stand.



Verwenden Sie dies, um sicherzustellen, dass Sie die richtige Datei ausgewählt haben. Wenn Sie sicher sind, dass Sie alles richtig ausgewählt haben Wenn Sie sicher sind, dass Sie alles richtig ausgewählt haben, können Sie abschließend auf die Schaltfläche "Uplaod" klicken. Sie werden zur Hauptseite weitergeleitet und die ausgewählte Datei wird hochgeladen. Vergewissern Sie sich, dass Sie unten auf der Seite ein kleines Kästchen sehen welches besagt 'File was uploaded succesfully'.



Beachten Sie, dass es nicht möglich ist, Dateien hochzuladen, die den gleichen Namen haben wie eine bereits existierende eine.


Erstellen einer Datei

Alternativ können Sie auch auf die Schaltfläche.

Wie der Name schon sagt, dient er zum Erstellen von .aml-Dateien direkt in der AML-NoSQL-Datenbank. Ein Klick auf auf die Schaltfläche wird Ihnen ein Fenster angezeigt, das dem Fenster zum Hochladen sehr ähnlich ist.



Wie beim Upload gibt es die Schaltflächen "Abbrechen" und "Hochladen" mit den gleichen Funktionalitäten wie beim Upload. Diesmal haben Sie eine Eingabezeile, in der 'Name' steht. Benutzen Sie diese, um einen Namen zu deklarieren, aber denken Sie daran, einen eindeutigen Namen zu wählen, weil sonst erhalten Sie eine Fehlermeldung. Wie Sie sehen können, wird auf der rechten Seite der Eingabezeile sehen können, müssen Sie nicht unbedingt .aml als Dateityp angeben. Die Software macht dies automatisch für Sie. Das große Textfeld unten dient dazu, Code für Ihre .aml-Datei zu schreiben. Es unterstützt sogar Syntax-Highlighting. Wenn Sie mit dem Schreiben fertig sind, können Sie durch Klicken auf die Schaltfläche Schaltfläche 'Hochladen'.



Sie werden auf die Hauptseite weitergeleitet und Ihre Datei wird in der Tabelle angezeigt. Stellen Sie erneut sicher, dass Sie die kleine Warnbox sehen, die Ihnen mitteilt, dass die Erstellung erfolgreich war.




Finden einer Datei

Wenn Sie mehrere Dateien in die Datenbank hochgeladen haben, kann das Auffinden bestimmter Elemente schwierig werden. Um Ihnen das Auffinden der gesuchten Dateien zu erleichtern, bietet AML-NoSQL eine Suchleiste am linken oben in der Datenbank eine Suchleiste, um Filterwörter anzugeben.



Sie können alle Arten von Schlüsselwörtern eingeben, zum Beispiel Zahlen, um nach dem Datum, der ID oder der Größe zu suchen oder Zeichen eingeben, um nach dem Namen der Datei zu suchen. Auch eine Kombination aus Zahlen und Buchstaben ist ist ebenfalls möglich.



So können Sie Dateien mit unerwünschten Attributen herausfiltern.


Verwendung der Tabelle

Die Tabelle ist das Herzstück der Hauptseite. Alle hochgeladenen .aml-Dateien werden hier angezeigt.




Dateien sortieren

Um sie leichter zu finden, können Sie die Tabelle sortieren. Dazu sollen Sie die Sortierleiste verwenden.



Dazu drücken Sie einfach auf eine der Kategorien am Tabellenkopf. Sie können aufsteigend oder absteigend entweder nach "ID", einer von der Software vergebenen Kennung zur Identifizierung hochgeladener Dateien, 'Name', das ist der Name, den Sie für Ihre Datei gewählt haben, 'Größe', das ist die Größe der von Ihnen Größe', die die Größe Ihrer Datei angibt, und 'Datum', das das Datum der Erstellung (nicht des Hochladens!) Ihrer Datei angibt. Zum Umschalten zwischen auf- und absteigend umzuschalten, klicken Sie einfach noch einmal auf das gewünschte Attribut.




Mit einem dritten Klick können Sie die Sortierung beenden.


Editieren, löschen oder eine Datei löschen

Der Punkt auf der rechten Seite der Tabellenüberschrift lautet "Aktionen".



Hier können Sie Dateien löschen, herunterladen oder bearbeiten. Um eine Datei zu bearbeiten, müssen Sie auf das Symbol klicken.
Sie werden dann zu einer Alertbox weitergeleitet, die wie die von "Datei erstellen" aussieht. Sie können nun den Namen ändern und den Code der Datei bearbeiten. Natürlich können Sie den Vorgang auch 'Abbrechen', wenn Sie keine Änderungen vornehmen möchten. Wenn Sie mit der Bearbeitung fertig sind, drücken Sie einfach auf 'Hochladen'.

Um Dateien herunterzuladen, können Sie diesen Button verwenden.
Der Download wird dann automatisch gestartet.

Die letzte Möglichkeit ist das Löschen von Dateien. Zum Löschen einer Datei drücken Sie die Schaltfläche .
Um sicherzustellen, dass Sie nicht versehentlich eine Datei löschen, erhalten Sie eine zweite Bestätigung bevor die Datei gelöscht wird.



Um den Löschvorgang zu bestätigen, müssen Sie auf "OK" klicken.


Seitenwechsel

In der rechten unteren Ecke sehen Sie die Paging-Vorgänge.



Aus Gründen der Benutzerfreundlichkeit und Performance werden nur 10 Elemente pro Seite angezeigt.



Auf jeden Fall können Sie diese Zahl auf bis zu 200 einstellen. Klicken Sie einfach auf die Zahl rechts neben 'Items per Seite' und wählen Sie die gewünschte Anzahl aus.



Auf der rechten Seite dieser Auswahl sehen Sie die Anzahl der Elemente, die pro Seite angezeigt werden. Zu guter Letzt sehen Sie in der rechten Ecke zwei Pfeile.



Sie können sie verwenden, um zwischen mehreren Tabellenseiten zu navigieren, damit Sie nicht zu viele Elemente pro Seite anzuzeigen.
